MINDSET 1

AI가 할 것 vs 내가 할 것

이 발표를 준비하면서 가장 먼저 생각한 것

AI가 할 것

PPT 디자인
PPT 제작
페이지별 컨텐츠 구성

내가 할 것

내용 목차 구성
스크립트 작성
완성본 검토 수정

WARNING

만약 이 분류 없이
시작한다면?

분류 후 진행

핵심(스크립트)에 집중
→ AI에게 나머지 위임
→ 검토만으로 완성

분류 없이 진행

목차부터 AI에게 부탁
→ "내 생각이랑 다른데?"
→ 결국 내가 다시 작업

각 단계마다 조금씩 AI 도움을 받는 방식은
가치 있는 일에 집중하지 못하고
AI에 의존적으로 업무를 하게 돼요.

결과가 기대와 다를 때 AI를 탓하게 되고
결국 다시 내가 처음부터 하는 상황이 됩니다.
